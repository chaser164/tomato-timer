Welcome to the Tomato Timer Chrome Extension!

This is an extension that will assist you with using the Pomodoro Technique and help you cut down on procrastination!

In order for this extension to work properly, please allow access all URLs (doing so will allow the timer to run properly behind the scenes).

Also, please pin this extension by clicking on the puzzle piece near the top-right corner of your browser for easy access to the extension's features.

USAGE NOTE: the timer will only count down while at least one Chrome tab is open.

Once pinned, click on the icon to open the popup and get started! I hope you have a productive day :)
